# SQL Mini Project – PostgreSQL | pgAdmin

## 🔍 Overview
This project explores advanced SQL concepts using PostgreSQL and pgAdmin, covering real-world sales and customer data scenarios. 

## 📊 Concepts Covered
- SQL Joins (INNER JOIN)
- Aggregate Functions (SUM, COUNT, AVG)
- Window Functions: 
  - RANK(), DENSE_RANK(), ROW_NUMBER()
  - LAG(), LEAD()
  - Moving Averages
  - Running Totals
- Date Functions
- Conditional Ranking
- Percent of Total

## 🧠 Key Business Questions Answered
- Total sales by region
- Top 5 customers by sales
- Rank products by sub-category
- Moving average of sales over time
- Percent contribution of each region
- Difference in sales between days

## 🛠️ Tools Used
- PostgreSQL
- pgAdmin
- SQL

## 📁 Files
- `queries.sql`: All SQL queries used in the project
